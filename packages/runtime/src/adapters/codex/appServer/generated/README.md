# Codex App Server Generated Protocol

This directory stores committed protocol artifacts generated by the installed Codex CLI for the Codex `app-server` stdio JSONL transport.

Generated with:

```bash
codex app-server generate-ts --out packages/runtime/src/adapters/codex/appServer/generated
codex app-server generate-json-schema --out packages/runtime/src/adapters/codex/appServer/generated/schema
```

Do not edit generated files by hand. Regenerate after upgrading Codex CLI.

Stable API only; experimental API is intentionally disabled for the runtime default.

Project helper:

```bash
npm run -w @aif/runtime codex:app-server:protocol:generate
```

Check committed artifacts against the installed CLI:

```bash
npm run -w @aif/runtime codex:app-server:protocol:check
```
